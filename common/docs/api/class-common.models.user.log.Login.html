Class Login

用户登录日志。因此“用户不存在”的日志并不记录在内。 记录如下内容： 1. 登录用户的 GUID ； 2. 登录时的 IP 地址； 3. 登录时间（以记录创建时间为准）； 4. 登录类型（如浏览器、移动、PC等）； 5. 登录结果（成功或失败）。 “用户不存在”错误记录在“系统”分类下。 此模型保存在 MongoDB 中。因此需要配置数据库名称和集合名称。
